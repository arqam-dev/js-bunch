## Installation

Using npm:
```shell
$ npm i -g npm
$ npm i js-bunch --save
```

In Node.js:
```js
// Load the full build.
var jb= require('js-bunch');

// Load method categories.
var array = require('js-bunch/array');
```

See the [package source](https://github.com/arqam-dev/js-bunch.git) for more details.
